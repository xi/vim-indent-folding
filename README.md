This plugin enables folding by indentation including the previous line in the
fold. This is ideal e.g. for nested lists:

    -   Item 1
        -   Item 1.1
        -   Item 1.2
    -   Item 2
        -   Item 2.1-----------------------------------------------------------
        -   Item 2.2
            -   Item 2.2.1
            -   Item 2.2.2
    -   Item 3-----------------------------------------------------------------
